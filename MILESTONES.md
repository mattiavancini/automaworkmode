# Milestones

## Milestone attuale: MVP operativo `0.1.1`

La versione corrente chiude un MVP deployabile e controllabile per uso operativo reale.

Funzionalita incluse:

- timer in minuti con default `120`
- range `1-720`
- selezione manuale persistente dei plugin
- plugin `protected` mostrati con checkbox `disabled`
- classificazione leggibile: `protected`, `recommended-heavy`, `neutral`, `unknown`
- pagina opzioni con stato Work Mode e inventario plugin
- widget rapido in Bacheca
- start/stop dalla pagina plugin
- notice persistente globale `Automa Modalita Operativa`
- countdown live nel notice admin
- countdown live nella pagina opzioni
- fallback server-side del tempo residuo al page load
- pulsante `Riattiva ora`
- lista verticale dei plugin spenti con nome e path
- fotografia iniziale completa del sistema plugin
- restore automatico
- fallback restore su admin load
- action link `Impostazioni` nella schermata Plugin
- descrizione plugin in italiano
